Career Management Application

 

Overview: This application will help professionals proactively manage their careers through a variety of services. Specifically targeting young professionals starting their careers and older professionals aiming to make a career switch, the application will provide the following services:

 

·      5-Year Plan and 10-Year Plan Templates

o   Users can populate these templates with their goals; these goals can be updated and users can add notes to track their progress

o   Application will alert the user when certain goals are approaching, haven’t been updated in a while, or need a significant amount of work to be achieved

·      Networking Tracker

o   If a user’s contacts are already using the application, the user will be able to create profiles of these contacts in a “Network Pool” that self-populate using the contacts’ LinkedIn information. Then, the user can add the following information:

§  Context of meeting

§  Strength of connection

§  Date of last meeting

§  Upcoming scheduled meetings

o   If the user’s contacts are not using the application, the user can still create profiles of these contacts in the “Network Pool” but will need to fill in the below basic information, in addition to the information above:

§  Name

§  Industry

§  Company

§  Title

o   Note: If the above profiles are too difficult to create, this information an also be stored in a basic spreadsheet-like format within the application

·      Article Center

o   Pulls in interesting articles related to workplace issues such as how to deal with difficult bosses or how to deal with fatigue resulting form long hours or staring at the computer screen all day

o   Users have the ability to “favorite” certain articles and can see the articles that their connections have favorited

·      Event Calendar

o   Allows users to enter in important events that are coming up, i.e. conferences or information sessions

o   The application will also suggest different events for users based on their profiles

 

Project Requirements:

·      About Me: Details from above will be included

·      User Accounts: Users will be required to create their own accounts; they will sign up using their LinkedIn profiles, which will provide the application with the following information:

o   Existing professional network

o   Profession

o   Industries of interest

o   Current and former employers
